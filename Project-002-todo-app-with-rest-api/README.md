# TODO Application

A simple todo application with the RESTful api using the gin

## Supported CRUD operations

- [ ] GET todo
- [ ] GET ALL todo

<br>

- [ ] CREATE todo
- [ ] CREATE multple todo

<br>

- [ ] UPDATE todo to done-un-done
- [ ] UPDATE ALL todo to done/un-done

<br>

- [ ] DELETE single todo
- [ ] DELETE all todo
